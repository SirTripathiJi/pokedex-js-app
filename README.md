# 🧾 Pokedex Web App

## 📌 Description
This is a responsive Pokedex web application built using HTML, CSS, and JavaScript. It fetches data from PokeAPI and displays Pokemon details dynamically.

## 🚀 Features
- Search Pokemon 🔍
- Filter and Sorting 📊
- Dark Mode 🌙
- Responsive UI 📱
- Pokemon Detail Page 📄
- Loader & Scroll to Top ⬆️

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- PokeAPI
